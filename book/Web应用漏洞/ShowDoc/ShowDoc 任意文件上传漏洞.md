# ShowDoc 任意文件上传漏洞

## 漏洞描述

ShowDoc 存在任意文件上传漏洞，攻击者通过构造特殊的数据包可以上传恶意文件控制服务器

## 漏洞影响

> [!NOTE]
>
> ShowDoc < V2.8.3

## FOFA

> [!NOTE]
>
> app="ShowDoc"
